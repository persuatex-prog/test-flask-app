# Hello API

A simple, lightweight Flask API that returns a greeting message. Built for learning, testing, or as a starting point for more complex APIs.

## 🚀 Features

- **Simple**: Single-file application with minimal dependencies
- **Fast**: Quick startup and response times
- **Free**: All open-source tools, no costs
- **Stateless**: No database required
- **Production Ready**: Can be deployed with Gunicorn

## 📋 Requirements

- Python 3.10 or higher
- pip (Python package manager)
